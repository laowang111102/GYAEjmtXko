# Python图书馆图书借阅管理系统 python1094

## 项目概述

本项目是基于Python的Django框架开发的图书馆图书借阅管理系统，旨在通过高效的后端架构为图书馆的日常借阅管理工作提供便捷、可靠的解决方案。

## 技术栈

- Python版本：3.11（建议）
- 开发框架：Django
- 开发软件：PyCharm
- 数据库：MySQL

## 功能模块

### 系统角色

- **管理员**
  - 个人中心
  - 用户管理
  - 图书馆管理
  - 图书信息管理
  - 图书类型管理
  - 借阅信息管理
  - 归还信息管理
  - 续借信息管理
  - 罚金信息管理
  - 留言板管理
  - 系统管理

- **用户**
  - 个人中心
  - 借阅信息管理
  - 归还信息管理
  - 续借信息管理
  - 罚金信息管理
  - 留言板管理
  - 我的收藏管理

## 运行环境

- 推荐在配备有Python 3.11环境的Linux或Windows服务器上运行。
- 需安装MySQL数据库服务器。

## 部署步骤

1. 安装Python 3.11环境。
2. 安装Django框架。
3. 安装MySQL数据库。
4. 克隆项目代码至本地。
5. 根据需求配置数据库连接。
6. 运行项目迁移以创建数据库表。
7. 启动项目服务。
